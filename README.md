 🏦 ATM Banking System   
 

Overview  


The ATM Banking System is a desktop application developed using **C#** and **Windows Forms**.  


It simulates the basic operations of a real Automated Teller Machine (ATM).  


The application allows users to securely log in, deposit money, withdraw cash, transfer funds, check their balance, and view transaction history.  


The project is built using **Object-Oriented Programming (OOP)** and follows a **Three-Tier Architecture**.  


Customer and transaction data are stored in **Excel files using EPPlus Need inslation**.  


This project was developed to improve desktop application development and software architecture skills.
